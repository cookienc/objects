## 객체 지향이란?

> 객체들을 메세지(상호작용)를 통해 서로 결합하고, 적절한 의존성을 같게 함으로써 변경에 용이한 설계를 만드는 것

## 추상클래스에서...

> 추상클래스에서 객체의 행동을 변경하면 상속된 객체들에게 영향을 미치지 않지만, 인스턴스 변수가 추가되면 상속된 객체들에게 영향을 미치게 된다.

-> 그럼에도 불구하고 추상클래스를 통한 변경의 최소화에 대한 이점이 있기 때문에, 추상클래스를 사용하는게 옳다.
